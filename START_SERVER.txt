ИНСТРУКЦИЯ ПО ЗАПУСКУ СЕРВЕРА

Для запуска этого веб-сайта доставки вам потребуется:

1. Установить PHP (версия 7.4 или выше)
2. Установить MySQL или MariaDB
3. Установить веб-сервер (Apache или Nginx)

СПОСОБ 1: Использование встроенного сервера PHP (для разработки)

1. Убедитесь, что PHP установлен на вашем компьютере
2. Откройте терминал/командную строку
3. Перейдите в директорию проекта: 
   cd /путь/к/вашему/проекту/workspace
4. Запустите встроенный веб-сервер PHP:
   php -S localhost:8000
5. Откройте браузер и перейдите по адресу:
   http://localhost:8000

СПОСОБ 2: Использование XAMPP/WAMP (для Windows) или MAMP (для Mac)

1. Скачайте и установите XAMPP с https://www.apachefriends.org/
2. Скопируйте всю папку проекта в папку htdocs (обычно находится в C:\xampp\htdocs\)
3. Запустите XAMPP Control Panel
4. Запустите Apache и MySQL
5. Откройте браузер и перейдите по адресу:
   http://localhost/название_вашей_папки/

СПОСОБ 3: Использование Apache с настройкой виртуального хоста

1. Скопируйте проект в папку веб-сервера (например, /var/www/html/)
2. Настройте виртуальный хост в Apache
3. Настройте базу данных MySQL
4. Запустите Apache и MySQL

НАСТРОЙКА БАЗЫ ДАННЫХ:

1. Запустите MySQL/MariaDB
2. Создайте базу данных с именем "delivery_by"
3. Импортируйте файл delivery_by (2).sql в базу данных
4. Настройте подключение в файле db.php при необходимости

Для обновления структуры базы данных после внесения изменений, запустите:
php update_db_structure.php

После запуска сервера вы сможете использовать все функции сайта:
- Регистрация и авторизация пользователей
- Калькулятор доставки
- Оформление заказов
- Страница оплаты
- Админ-панель
